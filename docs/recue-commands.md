## Purpose

Quick live checks and recovery commands for operation.

## Heartbeat Interpretation

The Heartbeat is now a multi-line HUD generated by the `Boardroom`:
- `🌍 GLOBAL | Pop: 24/60 | CPU: 12.5 (Bucket: 10k) | 💰 150k c (+2k)`
- `📋 QUEUE  | builder@W7N7 ➔ hauler@W7N7`: Shows the global, prioritized spawn queue sequence.
- `🚨 DEFENSE| Hostiles detected in: W9N6(1)`: Appears only when hostiles are present.
- `[LABEL] ROOM (RCL X | Phase) | NRG: x/y | Spawns: ACTION | TTL: Z`
- `  └─ ROLES:HAVE/NEED`: Shows the current population vs. the dynamically calculated need for that room.

## Audit Signals

- Tactical audit (every 200 ticks):
- `AUDIT-T <tick> | ENERGY <sum> | HOSTILES <sum> | SPAWN <busy>/<total>`
- Strategic audit (every 3600 ticks):
- `--- STRATEGIC AUDIT <tick> ---` plus per-room health lines.

## Console Checks

- Role distribution:
- `_.countBy(Game.creeps, c => c.memory.role)`

- Creeps assigned to a specific room:
- `_.filter(Game.creeps, c => c.memory.workRoom === 'W7N8' || c.memory.targetRoom === 'W7N8').map(c => c.memory.role)`

- Check a specific remote mining setup:
- `Memory.remoteRooms['W9N6']`

- Get current scout intelligence for a room:
- `Memory.inventory.rooms['W9N6']`

- Latest tactical audit snapshot:
- `Memory.audit && Memory.audit.tactical ? Memory.audit.tactical[Memory.audit.tactical.length - 1] : null`

- Latest strategic audit snapshot:
- `Memory.audit && Memory.audit.strategic ? Memory.audit.strategic[Memory.audit.strategic.length - 1] : null`

## Common Recovery Actions

- A creep is stuck:
- `Game.creeps['creep_name'].suicide()`

- Decommissioning obsolete/stuck creeps:
- Single creep: `Game.creeps['name'].memory.recycle = true`
- Mass recycle by role (e.g. idle scavengers):
  `Object.values(Game.creeps).filter(c => c.memory.role === 'scavenger').forEach(c => c.memory.recycle = true);`

- Start a new remote mining operation:
- `Memory.remoteRooms['W9N7'] = { base: 'W7N8', miners: 2, haulers: 2 }`

- Abandon a remote mining operation:
- `delete Memory.remoteRooms['W9N7']`
